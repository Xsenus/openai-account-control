# Индекс документации

Этот архив объединяет код проекта и всю подготовленную документацию в одном месте.

## Основной комплект документации

- `TZ.md` — базовое техническое задание.
- `UPDATE_CHECKLIST.md` — короткий чеклист обновления существующей установки.
- `MODULE_REFERENCE.md` — модульная карта проекта.
- `DEVELOPMENT_GUIDE.md` — руководство по запуску и разработке.
- `OPERATIONS.md` — эксплуатация и onboarding.
- `API.md` — описание backend API.
- `SECURITY.md` — модель безопасности.
- `DEPLOYMENT.md` — публикация в GitHub и автоматический деплой на VPS.
- `OpenAI_Account_Control_Center_TZ.docx` — DOCX-версия ТЗ.

## Дополнительный комплект расширенной документации

Папка: `ADDITIONAL_RU_DOCS/`

- `TECHNICAL_SPEC_RU.md` — расширенное техническое задание.
- `DEVELOPMENT_GUIDE_RU.md` — расширенная инструкция по разработке.
- `API_REFERENCE_RU.md` — дополнительное описание REST API.
- `CODE_WALKTHROUGH_RU.md` — подробный разбор модулей и функций.
- `VALIDATION_REPORT_RU.md` — отчет о проверке сборки.
- `OpenAI_Codex_Control_Center_TZ_i_Dokumentaciya.docx` — расширенный DOCX-комплект.

## Структура архива

```text
openai-account-control-center-complete/
├── backend/
├── frontend/
├── docs/
│   ├── ADDITIONAL_RU_DOCS/
│   └── ...
├── Makefile
└── README.md
```

## С чего начать

1. Откройте `README.md`.
2. Если нужно быстро обновить уже установленную систему, откройте `docs/UPDATE_CHECKLIST.md`.
3. Затем `docs/DEVELOPMENT_GUIDE.md`.
4. Для постановки задачи используйте `docs/TZ.md` и `docs/ADDITIONAL_RU_DOCS/TECHNICAL_SPEC_RU.md`.
5. Для детального разбора кода используйте `docs/ADDITIONAL_RU_DOCS/CODE_WALKTHROUGH_RU.md`.
